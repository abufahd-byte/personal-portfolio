# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, JavaScript, and PHP.  
The project is designed to showcase personal information, services, projects, blog content, and contact details in a clean and professional layout.

---

## 🚀 Features

- Responsive personal portfolio layout
- Homepage with professional introduction
- Services section
- Projects showcase
- Project details page
- Blog page
- Blog details page
- Contact page
- Clean and organized code structure
- Simple and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- Git
- GitHub
