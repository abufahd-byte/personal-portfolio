# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, JavaScript, and PHP.  
The project is designed to showcase personal information, skills, services, projects, and contact details in a clean and professional layout.

---

## 🚀 Features

- Responsive landing page
- About section
- Skills section
- Services section
- Projects showcase
- Contact section
- Clean and organized code structure
- User-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- Git
- GitHub
